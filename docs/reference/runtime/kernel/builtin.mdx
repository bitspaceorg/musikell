## Purpose

Thin re-export module that aggregates all builtin kernel submodules and provides the default `builtinRegistry`. Stateless kernels are pre-registered; stateful kernels must be created via their `mk*` constructors and registered separately during lowering.

**Module:** `Musikell.Runtime.Kernel.Builtin`

## Functions

### `builtinRegistry`

```haskell
builtinRegistry :: KernelRegistry
```

Registry pre-loaded with all stateless built-in kernels:

| Kernel                  | KernelId           |
| ----------------------- | ------------------ |
| `inputKernel`           | `"input"`          |
| `outputKernel`          | `"output"`         |
| `passthruKernel`        | `"passthru"`       |
| `nullKernel`            | `"null"`           |
| `gainKernel 1.0`        | `"gain"`           |
| `mixKernel`             | `"mix"`            |
| `clipKernel (-1.0) 1.0` | `"clip"`           |
| `channelMergeKernel`    | `"channel.merge"`  |
| `monoToStereoKernel`    | `"mono_to_stereo"` |

Stateful kernels (oscillators, delay, filter, dynamics, reverb, noise, envelope, sequencer, crossfade) are NOT included in the default registry. They require IO to initialize their internal state and must be created with their `mk*` constructors then registered via `registerKernel`.

## Usage Notes

- All builtin kernel modules require `{-# LANGUAGE OverloadedStrings #-}` because `KernelId` uses `Text`.
- The `Util` submodule (`Musikell.Runtime.Kernel.Builtin.Util`) provides shared helpers `forRange_` (strict loop) and `flushDenormals` (flush subnormal floats to zero) used across multiple kernels.
