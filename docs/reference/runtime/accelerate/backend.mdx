**Dependencies:** none

## Purpose

Stub module for the Accelerate GPU backend. Currently provides only a type and default constructor. Kernels fall through to CPU execution via the shared `KernelRegistry`. When Accelerate support lands, this module will marshal buffers to Accelerate arrays, run GPU computations, and marshal back.
