CLI layer: command-line argument parsing and dispatch.

The CLI follows a subcommand pattern: `musikell <command> [options] [args]`.

```
musikell run <file.mkl>                     execute a graph file
musikell run --block 512 <file.mkl>         execute with custom block size
musikell run --backend github:User/repo     execute with external backend
musikell config set key=value               set a config value
musikell config get key                     get a config value
musikell version                            show version information
musikell help                               show help
```
