Language layer: AST definition, hand-rolled recursive-descent parser, and lowering pass that transforms parsed programs into a runnable audio graph.

## Pipeline

```
.mkl source --> Parser --> AST (Program) --> Lowering --> Graph + ExecutionPlan + KernelRegistry
                  |                             |
             hand-rolled                  freshId, env, groups,
             precedence climbing          feedback edges, bypass/mute
```

## Modules

| Module                                  | Description                                                                  |
| --------------------------------------- | ---------------------------------------------------------------------------- |
| [AST](reference-language-ast)           | Complete type definitions for the Musikell abstract syntax tree              |
| [Parser](reference-language-parser)     | Hand-rolled recursive-descent parser with precedence climbing for binary ops |
| [Grammar](reference-grammar)            | Formal grammar specification for the `.mkl` DSL                              |
| [Lowering](reference-language-lowering) | AST-to-graph lowering: imports, bindings, groups, feedback, bypass/mute      |
