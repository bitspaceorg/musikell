Language layer: AST definition, hand-rolled recursive-descent parser, and lowering pass that transforms parsed programs into a runnable audio graph.

## Pipeline

```
.mkl source --> Parser --> AST (Program) --> Lowering --> Graph + ExecutionPlan + KernelRegistry
                  |                             |
             hand-rolled                  freshId, env, groups,
             precedence climbing          feedback edges, bypass/mute
```

## Modules

| Module                              | Description                                                                  |
| ----------------------------------- | ---------------------------------------------------------------------------- |
| [AST](language/ast.mdx)             | Complete type definitions for the Musikell abstract syntax tree              |
| [Parser](language/parser.mdx)       | Hand-rolled recursive-descent parser with precedence climbing for binary ops |
| [Grammar](../reference/grammar.mdx) | Formal grammar specification for the `.mkl` DSL                              |
| [Lowering](language/lowering.mdx)   | AST-to-graph lowering: imports, bindings, groups, feedback, bypass/mute      |
