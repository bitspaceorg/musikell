IO layer: Unix pipeline streaming for real-time audio.

The IO layer connects stdin → graph → stdout in a single streaming loop using Conduit.

## Pipeline

```
stdin → sourceStdin/chunkedSource → bytesIntoBuffer → executeBlock → unsafeBufferToBytes → sinkStdout → stdout
         (StreamInput)              (zero-alloc)       (Scheduler)    (zero-copy)           (StreamOutput)
```

## Modules

| Module                                           | Description                                             |
| ------------------------------------------------ | ------------------------------------------------------- |
| [StreamInput](reference-io-stream-input)         | PCM reads from stdin into pre-allocated buffers         |
| [StreamOutput](reference-io-stream-output)       | Buffer to PCM bytes with safe and zero-copy variants    |
| [ConduitPipeline](reference-io-conduit-pipeline) | Complete streaming pipeline: stdin → scheduler → stdout |
