## Quick Reference

- **Build:** `nix build`
- **Dev shell:** `nix develop '.#dev'`
- **Unit tests:** `nix build .#checks.aarch64-darwin.dev-unit`
- **Format:** `nix fmt`
- **Must `git add` new files** before `nix build` (Nix flakes only see tracked files)
- **GHC:** 9.8.4, Haskell2010
